# 30MinScript
심심할 때 30분 동안 자동화 스크립트 짜기

- 간단하게는 .bat부터 python이나 node를 사용할 것 같습니다.
- 따로 설치해야되는건 그때 그때 적어둡시다.
- 최근에 컴퓨터로 2번 이상했던 작업을 떠올립니다.
- 하루 30분씩 투자해서 조금씩 자동화를 해봅시다.
- 현실에서 자동사냥을 돌려봅시다.
- 에제를 동봉합시다.

## 새로운 폴더 만들기
```
30MinTemplate.bat <dir name>
```
